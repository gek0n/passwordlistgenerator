--delimiter
		Use this option to specify delimiter, which should be paste between elements in combined string. By default it has <space> value.
		Example:
			PasswordListGenerator comb -i textFile.txt --delimiter a

		<-*textFile content*->
			0
			1
			2

		Output:
			0a1
			0a2
			1a0
			1a2
			2a0
			2a1

		If you want to use not alphanumeric symbols as delimiter, then you better should use "" (double quotes) to specify the symbols.
		Example:
			PasswordListGenerator comb -i textFile.txt --delimiter="<|>"
